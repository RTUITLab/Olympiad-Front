Общие положения о проведении тестирования

Решение каждого задания - это файл с исходным кодом программы, объемом не более 5 МБ. Исходный код будет скомпилирован и выполнен с различными наборами входных данных. Если отправленное решение отработает некорректно, то система выдаст соответствующее сообщение.

Время на выполнение программы - 1 секунда. Превышение временного лимита автоматически приводит к состоянию "Решение выполнялось слишком долго"

Индикаторы отправленного решения

Ошибка при компиляции

Ошибка во время выполнения

Неверный результат работы

Решение выполнялось слишком долго

Решение в очереди на проверку

Решение проверяется

Решение верно

Пример правильно решенного задания

### Задача инкремент > На вход программе подается одно целое число A. Необходимо вывести число, равное A + 1; #### Ограничения входных параметров >231 ≥ A ≤ 231-2 #### Правильное решение на языке Pascal (используется Pascal ABC) ``` var val: Integer; begin Readln(val); WriteLn(val + 1); end. ```

Внимание!

Для отправки задания текст кода должен находиться в файле, например "Program.pas". После отправки программа отобразится в таблице отправленных решений и с течением времени изменит свой статус в следующем порядке:

Решение в очереди на проверку

Решение проверяется

Решение верно
