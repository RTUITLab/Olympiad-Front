## Общие положения о проведении олимпиады > Как решение каждого задания необходим файл с исходным кодом программы. Время на выполнение программы - 1 секунда. Если программа работает дольше - задание считается невыполненным

Индикаторы отправленного решения:

Ошибка при компиляции программы
Ошибка во время выполнения программы
Неверный результат работы программы
Решение в очереди на проверку
Решение проверяется
Решение верно
