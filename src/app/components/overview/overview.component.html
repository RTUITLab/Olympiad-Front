Общие положения о проведении тестирования

Как решение каждого задания необходим файл с исходным кодом программы.

Исходный код при необходимости будет скомпилирован, и выполнен с различными наборами входных данных. При некорректной работе программы данному отправленному решению будет присвоен соответствующий статус.

Время на выполнение программы - 1 секунда. Если программа работает дольше - задание считается невыполненным, и ему присваевается статус "Неверный результат работы программы"

Индикаторы отправленного решения:

Ошибка при компиляции программы

Ошибка во время выполнения программы

Неверный результат работы программы

Решение в очереди на проверку

Решение проверяется

Решение верно

Пример правильно решенного задания

### Задача инкремент > На вход программе подается одно целое число A. Необходимо вывести число, равное A + 1; #### Ограничения входных параметров >231 ≥ A ≤ 231-2

Правильное решение на языке Pascal (используется Pascal ABC)


      var
        val: Integer;

      begin
        Readln(val);
        WriteLn(val + 1);
      end.

Данный текст программы необходимо вложить при отправке задания, внутри файла, например Program.pas

После потравки решения оно появится в таблице отправленных решений и с течением времени изменит свой статус в следующем порядке

Решение в очереди на проверку

Решение проверяется

Решение верно
